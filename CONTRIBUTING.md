## Código de Conducta

Este proyecto y todos los participantes se rigen por nuestro [Código de Conducta](CODE_OF_CONDUCT.md). Al participar, se espera que mantengas este código. Reporta comportamiento inaceptable según se describe en el documento.

## ¿Cómo Contribuir?

### 1. Reportar Bugs
- Usa el [rastreador de issues de GitHub](https://github.com/enriqueherbertag-igtm/quantum-flux-bridge/issues)
- Verifica si el bug ya ha sido reportado
- Incluye título descriptivo y descripción clara
- Proporciona pasos para reproducir
- Especifica comportamiento esperado vs actual
- Incluye versión del proyecto, sistema operativo, etc.

### 2. Sugerir Mejoras o Características
- Abre un Issue con la etiqueta `enhancement`
- Describe el problema que resuelve la mejora
- Explica por qué sería útil para el proyecto
- Si es posible, propone una implementación aproximada

### 3. Contribuir con Código
1. **Fork** el repositorio
2. **Crea una rama** para tu feature/bugfix:
   ```bash
   git checkout -b feature/nueva-funcionalidad
   # o
   git checkout -b fix/correccion-bug
